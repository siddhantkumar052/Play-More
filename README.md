## How to Run

1. Install
npm: [https://nodejs.org/en/](https://nodejs.org/en/)

```bash
npm install npm@latest -g
```
2. Install the npm packages
```bash
npm install
```


3. Seed file
```bash
node seed
```

4. Run the application
```bash
npm start
```
